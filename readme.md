# Hexo Brain - 一款blog agent



项目简述

Hexo Brain 是一个基于静态博客的轻量级 RAG（Retrieval-Augmented Generation）系统。项目采用前后端分离架构：前端依托 Hexo 静态页面嵌入聊天组件，后端基于 FastAPI 构建 Agent 服务，实现问题 embedding、向量检索与上下文拼接，并调用 LLM API 生成回答。通过对博客文章进行切块与向量化，构建本地语义索引，实现低成本、高可控的知识增强问答系统。



简单逻辑：

```latex
静态博客 (GitHub Pages)
          +
轻量 Agent API
          +
本地向量 JSON
          +
LLM API
```
